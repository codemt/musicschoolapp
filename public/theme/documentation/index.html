Adding Blog Posts #back to top

In the main tag you as you can see there is article tag. Each article tag means each post just copy and paste like we did. And edit contents like img url, title, content. You can remove thumbnail if you want. We coded this theme modular and clean!

  1. Open file in code editor.
  2. Go inside of <main> tag.
  3. You will see posts are commented as it's name.
  4. Customize as you want!
  5. Don't forget to save before close.

Adding Widgets #back to top

We're provide you a lot of customization options if you have little bit HTML or CSS information.

  1. Open file in code editor.
  2. Go widget area which you want to change.
  3. You will see widgets are commented as it's name.
  4. Customize as you want!
  5. Don't forget to save before close.
